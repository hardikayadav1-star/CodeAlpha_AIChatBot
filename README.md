# 🤖 AI Chatbot using Java

A Java-based AI Chatbot developed for the CodeAlpha Internship. The chatbot uses NLP-based keyword matching and a Java Swing GUI to provide real-time responses to user queries.

## ✨ Features

- Interactive Chat Interface
- NLP-Based Text Processing
- FAQ Knowledge Base
- Real-Time Responses
- Chat History Saving
- Java Swing GUI

## 🛠 Technologies Used

- Java
- Java Swing
- OOP Concepts
- NLP Basics
- Git & GitHub

## 🚀 How to Run

1. Open the project in IntelliJ IDEA.
2. Run `ChatBotGUI.java`.
3. Start chatting with the bot.
